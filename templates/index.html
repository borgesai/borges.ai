welcome

Borges is your new friendly librarian. Keep track of books you've read or want to read.

It's like Goodreads. But built for 2020.

features

  • - import data from Goodreads
  • - synchronize data with Goodreads
  • - no ads
  • - clean interface
  • - fast load times
  • - works on mobile
  • - focus on text and not images. Distraction-free interface
  • - custom domain names
  • - markdown support
  • - auto backup data to Dropbox. You own your data!
  • - social reading (coming soon)
  • - API (coming soon)
